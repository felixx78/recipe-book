# Recipe book (Test task)

## Tech Stack

**Client:** Next.js, Tailwind

**Server:** Nest.js

## Run Locally

Clone the project

```bash
  git clone https://github.com/felixx78/recipe-book
```

Go to the project directory

```bash
  cd recipe-book
```

Install dependencies and run client

```bash
  cd ./frontend
```

```bash
  yarn install
```

```bash
  yarn dev
```

Install dependencies and run server

```bash
  cd ./server
```

```bash
  yarn install
```

Run server

```bash
  yarn start:dev
```
